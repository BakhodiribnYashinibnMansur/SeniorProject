Definition: f(n) = Ω(g(n)) if there exist constants c > 0 and n₀ ≥ 0 such that f(n) ≥ c · g(n) for all n ≥ n₀.

Use the sliders to adjust c and n₀. The visualization shows whether f(n) stays above c · g(n) for all n ≥ n₀. If it does, the Ω relationship holds with those constants.
