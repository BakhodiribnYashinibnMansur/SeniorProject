How it works: Both bubble sort and selection sort use two nested loops, producing O(n²) comparisons. The outer loop (i) runs n times, and for each iteration the inner loop (j) scans remaining elements. Watch the comparison counter grow quadratically as the animation proceeds. For n=15, the maximum comparisons are 15×14/2 = 105.
