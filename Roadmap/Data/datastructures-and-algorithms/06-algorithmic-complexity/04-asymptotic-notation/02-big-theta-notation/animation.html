How to use: Select f(n) and g(n), then adjust the constants c₁, c₂, and n₀ using the sliders. The visualization shows whether f(n) stays "sandwiched" between c₁·g(n) (red, lower bound) and c₂·g(n) (green, upper bound) for all n ≥ n₀. If the sandwich holds, f(n) = Θ(g(n)) with those constants. The shaded blue region highlights where the bounds are active. Try picking mismatched f(n) and g(n) (e.g., f(n) = n³ with g(n) = n) to see when no valid constants exist.
